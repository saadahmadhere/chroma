Avatars

Avatars are used for display pictures, images in cards, email signatures, etc.

You can use avatar of 4 different sizes. Use class avatar and avatar_circle with the img tag for circular avatars. For size, use avatar_xs, avatar_sm, avatar_md and avatar_lg.

You can use avatars for texts too. Use classes avatar_text in addition to the previous classes. Use the div tag here.
